## 题目思路

还是统计字母个数。和有效的字母异位词是类似的思路。

## 代码实现

```python
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        letters = dict()
        for char in magazine:
            if letters.get(char) is None:
                letters[char] = 1
            else:
                letters[char] += 1
        for char in ransomNote:
            if letters.get(char) is None:
                return False
            else:
                letters[char] -= 1
                if letters[char] < 0:
                    return False
        return True
```
