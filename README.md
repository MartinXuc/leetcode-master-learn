# LeetCode Master Learn

面向算法与数据结构刷题的学习项目，主要参考与复盘「代码随想录算法训练营」，按章节与每日任务组织题解与代码，实现以 Python/C++ 为主。

## 项目内容

- 覆盖数组、链表、哈希表、字符串等核心专题，配套题解与示例代码。
- 每日任务安排与学习笔记，便于循序渐进地完成训练营计划。
- 以实践为导向：每题包含「题目思路」「代码实现」「看录后想法/小结」。

## 目录结构

- `题目文档/`：按章节整理的题解与代码，例如数组、链表、哈希表、字符串等。每题通常包含同名 `.md` 文档与示例代码（如 `code.YYYYMMDD.py`）。
- `每日任务/`：每日刷题计划与链接说明，如 `day 7 哈希表 part02.md`。
- `博客文档/`：训练营按天记录的博客式总结与思考。
- `代码随想录 原文/`：参考原文与链接。
- 根目录示例：`demo.py`、`demo.cpp`、`README.en.md`、`LICENSE`。

## 使用方式

- 从 `每日任务/` 选择当天计划，按链接阅读题目与讲解后，在 `题目文档/` 查阅对应题解与代码。
- 直接按专题学习：进入 `题目文档/` 的某一章，依序阅读与实践。

## 运行示例

- Python：确保本地安装了 Python 3，进入相应目录运行示例代码，例如：
  - `python 题目文档/第 3 章 哈希表篇/5. 两数之和/code.20251206.py`
- C++：使用本地编译器编译运行示例文件，例如：
  - `g++ demo.cpp -std=c++17 -O2 -o demo && ./demo`

## 命名约定

- 代码文件常以 `code.YYYYMMDD.py` 命名，表示解题日期。
- 文档以「章节/题目名.md」组织，便于检索与复盘。
